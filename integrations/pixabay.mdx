## Account Creation

<Steps>
  <Step title="Visit Pixabay's Website">
    Go to [pixabay.com/api/docs](https://pixabay.com/api/docs/) and click "Join" in the top right corner.
    
    <img src="/images/pixabay/homepage.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Choose Signup Method">
    * Sign up with email/password.
    * Or continue with social media authentication.
    
    <img src="/images/pixabay/signup.png" style={{ borderRadius: '0.5rem' }} />
  </Step>
</Steps>

## Generate API Key

<Steps>
  <Step title="Access API Settings">
    1. Log in to your Pixabay account.
    2. Navigate to the API section in your account settings.
    
    <img src="/images/pixabay/api-settings.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Create New API Key">
    1. Click "Create API Key."
    2. Name your key (e.g., "Create Integration").
    3. Click "Generate."
    
    <img src="/images/pixabay/apigen.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Copy Your Key">
    1. Copy the generated API key.
    2. Store it securely - you won't be able to see it again.
    
    <Warning>
      Keep your API key secure - it grants access to your account data.
    </Warning>
  </Step>
</Steps>

## Add to Create Project

<Steps>
  <Step title="Add the Integration">
    1. In your Create project, type `/` in chat.
    2. Select "Pixabay" from the menu.
  </Step>

  <Step title="Add Your API Key">
    1. Paste your API key into "PIXABAY_API_KEY."
    
    <Tip>
      Need new credentials? Regenerate in the Pixabay dashboard.
    </Tip>
  </Step>

  <Step title="Start Using">
    You can now use prompts with the integration to access images and videos.

    Example prompts:
    * `Search for images of cats using /Pixabay`
    * `Get a random image using /Pixabay`
    * `Create a gallery app with /Pixabay`

    <Warning>
      Pixabay API has rate limits - implement error handling for best reliability.
    </Warning>

    Example prompt for optimization:
    ```
    When using /Pixabay, cache the image results in my database
    ```
  </Step>
</Steps>

<Note type="success">
  Your Pixabay integration is ready! Use `/Pixabay` in chat to access images and videos.
</Note>
