## Generate API Key

<Steps>
  <Step title="Access Your Dashboard">
    1. Log in to your Perplexity account
    2. Navigate to the API section
    
    <img src="/images/perplexity/dashboard.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Create API Key">
    1. Click "Generate New Key"
    2. Name your key (e.g., "Create Integration")
    
    <img src="/images/perplexity/apigen.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Copy Your API Key">
    1. Copy the generated API key
    
    <Warning>
      Keep your API key secure - it grants access to your Perplexity account and quota
    </Warning>
  </Step>
</Steps>

## Add to Create Project

<Steps>
  <Step title="Add the integration">
    1. In your Create project, type `/` in the chat 
    2. Select "Perplexity" from the menu
  </Step>

  <Step title="Add your API key">
    1. Paste your API key into the "PERPLEXITY_API_KEY" field
    
    <Tip>
      Need a new key? Generate one in your Perplexity dashboard.
    </Tip>
  </Step>

  <Step title="Use it">
    You can now use prompts with the integration to access AI research capabilities.

    Some examples:
    * `Research this topic using /Perplexity`
    * `Answer complex questions with /Perplexity`
    * `Create a research assistant using /Perplexity`

    <Warning>
      API has rate limits based on your plan - implement caching for frequently accessed queries.
    </Warning>

    Example prompt for optimization:
    ```
    When using /Perplexity, cache the research results in my database
    ```
  </Step>
</Steps>

<Note type="success">
  Your Perplexity integration is now ready! Use the `/Perplexity` command in chat to access AI research capabilities.
</Note>
