Before you begin, you will need an account on [Pexel&#x73;**.**](https://www.pexels.com/)

## Account Setup

Visit [Pexels](https://www.pexels.com/)

On their home page click on “Join” in the upper right

<img height="303" width="624" src="https://lh7-rt.googleusercontent.com/docsz/AD_4nXegFRdrCRgWAd6FltXOQHSM_p9amztFsQQuGzAEJbKOoGCyPfBk-GLybWWpz7ZeluqCJo9KAi7UjOkC9L_WtcLeC104f5IW8eG-J-WCRTr4UPjmC45vHeArqGDX_yQGhirJOZGj?key=NZ4d7cCGppQiJdZwRfiRd3BA" />

* Sign up with your email or authenticate with Google

## ![](/image.png)Generate API Token for Create

* Click on the three dot menu to the left of your Avatar/Initials on the Pexel Landing Page

![](/Screenshot2025-02-03at12.47.42PM.png)

* Navigate to “Images & Video API”

Click on the “Your API Key” button

![](/Screenshot2025-02-03at12.50.50PM.png)

## Create New Token

Create a new API token for your Create project

![](/Screenshot2025-02-03at12.52.30PM.png)

* Copy Token

Make sure to copy your API Key. You may revisit this API key later by returning to your Pexel dashboard at anytime

## Add to Create

Enter and save your API token in Create inside the Pexel prompt from the builder.&#x20;

Remember to keep your API token secure and never share it publicly. If you need a new API token, please contact [api@pexels.com](mailto:api@pexels.com)
