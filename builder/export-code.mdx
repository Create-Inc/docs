## Export Code

Navigate to the Code tab and click Download Project to get your exported project zip file.

![Code Download](/images/codedownload.png)

<Note>
  You can use the exported code in our [App Compliance Check](/builder/mobile/testing#apple-app-compliance-check) to ensure your app meets Apple's guidelines before App Store submission.
</Note>
