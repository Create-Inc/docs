</Note>

## Account Setup

<Steps>
  <Step title="Visit Replicate">
    Go to [replicate.com](https://replicate.com/) and click "Get started"
    <img src="/images/replicate/replicate-get-started.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Sign in with GitHub">
    Click "Sign in with Github" to continue
    <img src="/images/replicate/replicate-github-signin.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Authorize Access">
    Allow Replicate to access your GitHub account
    <img src="/images/replicate/replicate-authorize.png" style={{ borderRadius: '0.5rem' }} />
  </Step>
</Steps>

## Configure Billing

<Steps>
  <Step title="Open Account Settings">
    Navigate to "Account settings" in the menu
    <img src="/images/replicate/replicate-account-settings.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Access Billing">
    Click on the "Billing" section
    <img src="/images/replicate/replicate-billing-menu.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Set Up Payment">
    Click "Set up billing" to continue
    <img src="/images/replicate/replicate-setup-billing.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Add Payment Method">
    Enter your payment information and click "Save card"
    <img src="/images/replicate/replicate-save-card.png" style={{ borderRadius: '0.5rem' }} />
  </Step>
</Steps>

## Generate API Token

<Steps>
  <Step title="Access API Tokens">
    Navigate to "API tokens" in the settings menu
    <img src="/images/replicate/replicate-api-tokens.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Create New Token">
    Create a new API token for your Anything project
    <img src="/images/replicate/replicate-create-token.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Copy Token">
    <Warning>
      Make sure to copy your token now - you won't be able to see it again!
    </Warning>
    <img src="/images/replicate/replicate-copy-token.png" style={{ borderRadius: '0.5rem' }} />
  </Step>

  <Step title="Add to Anything">
    Enter and save your API token in Anything
    <img src="/images/replicate/replicate-save-token.png" style={{ borderRadius: '0.5rem' }} />
  </Step>
</Steps>

<Note>
  Remember to keep your API token secure and never share it publicly. If you need to generate a new token, you can revoke the old one from the Replicate dashboard.
</Note>
