1. [QUICK TIPS](/create/quick-tips)

In Create, it's possible to connect to any external API using functions. For a full, step-by-step guide on how to do this, check our [functions docs](https://docs.create.xyz/create/builder-add-ons/functions#external-apis).

Or follow along with this YouTube video tutorial, where we'll connect the [Exa AI Search API](https://exa.ai/) to Create.

* Make a new function in Create

* Go to the Exa docs

* Generate an API key

* Add this API key as a secret to your Create function

* Copy to Search docs and CURL request format

* Paste it into your Create prompt

* Click generate

* Test the API connection

* Add a new page in Create and build your frontend
